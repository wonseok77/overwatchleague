# OW League - Team Progress
# Last updated: 2026-03-05

## Architecture Decisions
- Monorepo: frontend/ + backend/ in single repo
- DB: PostgreSQL 16 via Docker
- Auth: JWT (HS256, 7-day expiry)
- API proxy: Vite dev server proxies /api to backend:8000
- File uploads: local filesystem at /uploads/screenshots/{match_id}/
